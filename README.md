## Quick Start

1. **Clone or fork this repository**
2. **Customize the content** in `index.html`:
   - Update the hero title and subtitle
   - Add your own about description
   - Update skills with your technologies
   - Replace project placeholders with your actual projects
   - Update contact links with your email, GitHub, and LinkedIn
3. **Deploy to GitHub Pages**:
   - Go to your repository Settings
   - Navigate to Pages section
   - Select the branch to deploy (usually `main` or `master`)
   - Save and wait for deployment

## Customization Guide

### Personal Information
Edit `index.html` to update:
- Hero section: Your name and tagline
- About section: Your description and background
- Skills section: Your technical skills
- Projects section: Your projects with descriptions and links
- Contact section: Your email, GitHub, and LinkedIn URLs

### Colors and Styling
Edit `styles.css` to customize:
- Color scheme (CSS variables in `:root`)
- Fonts and typography
- Spacing and layout
- Animation effects

### Additional Features
Edit `script.js` to modify:
- Navigation behavior
- Scroll animations
- Interactive elements
